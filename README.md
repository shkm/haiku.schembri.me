# haiku

A minimal haiku blog. Requires [Pandoc](https://pandoc.org/) and optionally [ImageMagick](https://imagemagick.org/) for OG images.

## Usage

```sh
./haiku new "Title"       # create a new haiku
./haiku build             # build the site to dist/
./haiku start             # build, serve, and watch for changes
```

Edit the generated file in `posts/`, write three lines, done.
